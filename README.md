# SkinGlow

Simple skincare e-commerce website using:
- Vanilla JavaScript
- Node.js
- Express.js
- MongoDB
- Midtrans Snap

---

## Install Backend

```bash
cd backend
npm install
```

---

## Setup Environment

Copy:

```bash
.env.example
```

becomes:

```bash
.env
```

Fill all credentials.

---

## Run Backend

```bash
npm run dev
```

---

## Seed Products

```bash
npm run seed
```

---

## Run Frontend

Open:

```bash
frontend/index.html
```

or use Live Server.
